



# Skills

## Create/update ERD

1. Read h5media/models.py to determine database tables and relationships
2. Create a in Graphviz's DOT language.  Save this diagram as docs/erd.dot
   1. Use html tables as the node labels.
      1. The top row should have a light blue background and the text should be in bold
      2. Table should have cellborder=1 cellspacing=0
   2. Use ortho splines
3. Render the ERD as docs/erd.svg
